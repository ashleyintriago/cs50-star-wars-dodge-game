# cs50-star-wars-dodge-game
CS50 Week 0 Scrath project focused on game logic, event-driven programming, and progressive difficulty.

## Features
- Millenium Falcon playar
- Dodge Darth Vader obstacles
- Incresing difficulty
- Timer and high score system
- Intro screen
- Custom game over dialogue

## Concepts Practiced
- Event-driven programming
- Loops
- Conditionals
- Variables
- Basic game design
